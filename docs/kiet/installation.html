Install Ubuntu on Windows

Set up Windows Subsystem for Linux (WSL2) to run Ubuntu Linux natively.

Phase 1: Environment Verification

1. Open PowerShell
Click Start, search for "PowerShell," right-click it, and select Run as Administrator.
2. Check for WSL
PowerShell 
wsl --list --online
üí° What to expect:
  • If the command works: Skip to Phase 3
  • If "command not found": Continue to Phase 2

Phase 2: Enabling Required Features

3. Enable WSL
PowerShell 
dism.exe /online /enable-feature /featurename:Microsoft-Windows-Subsystem-Linux /all /norestart
3. Enable Virtual Machine Platform
PowerShell 
dism.exe /online /enable-feature /featurename:VirtualMachinePlatform /all /norestart
4. Restart & Verify
Restart your computer. Open PowerShell as Administrator and run wsl --list --online to verify.

Phase 3: Install Ubuntu

5. Install Ubuntu 24.04
PowerShell 
wsl --install Ubuntu-24.04

Phase 4: User Configuration

Create Username
Once Ubuntu launches, you'll be prompted to create a new user account. Enter a username (use lowercase letters only, no spaces). For example: john or student
Set Password
After entering your username, you'll be asked to create a password. Choose something memorable like your birth year (e.g., 2005) or a simple password you won't forget.
‚ö†Ô∏è IMPORTANT - Password Security Behavior:
  • Your password will NOT be visible as you type - this is a security feature, not a bug
  • You will see NO characters, dots, or asterisks appearing on screen while typing
  • The cursor will NOT move as you type your password
  • This is completely normal Linux behavior designed to protect your password
  • Simply type your password carefully and press Enter when done
  • You'll be asked to type it twice to confirm - both times it will be invisible
‚úÖ Tips for Success:
  • Type slowly and carefully since you can't see what you're typing
  • Don't panic if nothing appears - this is expected behavior
  • If you make a mistake, press Backspace a few times and start over
  • Remember your password - you'll need it for installing software later

Download & Configure Tools

Download the setup scripts, unzip them, and execute the automation tools.

Step-by-Step Configuration

1. Download the Zip File
Use curl to download the setup_scripts.zip file.
Bash 
curl -L -o setup_scripts.zip https://github.com/aikaryashala/bootcamp/raw/b4a8b253eba8a70c1ee282a1b2b47cedac588bdc/docs/kiet/setup_scripts.zip

Unzip the folder and enter the directory.
Bash 
unzip setup_scripts.zip && cd setup_scripts
3. Run Setup Tools
This script installs clang, lldb, micro, and other development essentials.
Bash 
python3 setup_tools.py
4. Run AIKBT Installer
This configures the custom debugger interface.
Bash 
python3 install_aikbt.py
5. Apply Changes
Bash 
source ~/.bashrc

Ensure all tools are correctly installed by checking their versions.

2. Compile and Debug Workflow

Bash

Compile the Code
Compile sum_max.c using clang with debug flags (-g) and no optimization (-O0).
Bash

Start Debugger (LLDB)
Bash

‚úÖ Check:

Run the following commands inside LLDB to set a breakpoint, run the program, and view the visual backtrace.
LLDB

(lldb) run
(lldb) aik bt

Contact QR Code Generator

Install and configure the Contact QR Code Generator to create shareable contact information.

Setup QR Code Generator

1. Install QR Library and Download Script
Install the QR code generation library with image support and download the contact QR generator script from GitHub.
Bash 
pip install qrcode[pil] && curl -O https://raw.githubusercontent.com/sandeepbangaru17/qr/main/contact_qr.py
üí° What this does:
  • Installs the qrcode library with PIL (Python Imaging Library) support
  • Downloads the contact_qr.py script to your current directory
  • Prepares your environment to generate QR codes for contact sharing
2. Run the QR Code Generator
Execute the script to generate contact QR codes.
Bash 
python3 contact_qr.py
‚úÖ Expected Output:
  • The script will prompt you to enter contact information (name, phone, email, etc.)
  • A QR code image file will be generated in your current directory
  • You can scan this QR code with any smartphone to quickly save contact details
  • Perfect for sharing your contact information at events or in your portfolio

Reel Generator

Setup and run the Reel Generator to create engaging video content from your learnings.

Install and Run Reel Generator

1. Download and Execute Setup Script
This command downloads and runs the Reel Generator setup script directly from GitHub.
Bash 
curl -fsSL https://raw.githubusercontent.com/ppavankumar19/Reel_Generator/refs/heads/main/setup_reels.py | python3
üí° What this does:
  • Downloads the Reel Generator setup script
  • Installs all necessary dependencies automatically
  • Configures the video generation environment
  • Starts a local web server on port 8000
2. Access Your Generated Video
Once the script completes, open your web browser and access the generated video.
Browser URL 
http://localhost:8000/videos/my-learnings.mp4
‚úÖ How to use:
  • Copy the URL above
  • Open any web browser (Chrome, Firefox, Edge)
  • Paste the URL in the address bar and press Enter
  • Your generated learning reel video will play directly in the browser
  • You can download the video or share it on social media
3. Alternative: Access Using Ubuntu IP (If localhost doesn't work)
If the localhost URL doesn't work, you can access the video using your Ubuntu machine's IP address.
First, find your Ubuntu IP address:
Bash 
hostname -I
üí° Understanding the output:
  • The command will display one or more IP addresses
  • Use the first IP address shown (usually starts with 172.x.x.x or 192.168.x.x)
  • Example output: 172.24.144.1 192.168.1.100 ‚Üí use 172.24.144.1
Then, use this URL format in your browser:
Browser URL Format 
http://<ubuntu-ip>:8000/videos/my-learnings.mp4
‚ö†Ô∏è Replace <ubuntu-ip> with your actual IP:
  • If your IP is 172.24.144.1, use: http://172.24.144.1:8000/videos/my-learnings.mp4
  • If your IP is 192.168.1.100, use: http://192.168.1.100:8000/videos/my-learnings.mp4
  • Make sure to replace <ubuntu-ip> with the IP address from the hostname -I command
